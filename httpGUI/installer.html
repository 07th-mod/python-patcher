Get Installer and Game Log (Windows)

There are two logs:
- The installer logs (produced by this installer). This may be useful if your install didn't fully complete or seems stuck.
- The game log (produced by the game). This may be useful if your game freezes or crashes.

The button below will generate a .zip file containing both of these logs. Please note that the game log only records one session, so make sure you have just recreated the bug in the most recent session.

You should submit this to our Discord in the Umineko #umi_support or Higurashi #higu_support channel when asking for help.
